# Research-Agent

Autonomous research agent that searches 10+ sources in parallel (Web, Wikipedia, arXiv, Semantic Scholar, GitHub, Hacker News, Stack Overflow, Reddit, YouTube) and synthesizes results into a structured report.

## Getting started

1. **Get a free API key** from [Groq](https://console.groq.com) (takes 2 min, no credit card)
2. Paste the key in the sidebar **LLM API Key** panel and click **Apply**
3. Type a research topic and click **Iniciar Investigación**
